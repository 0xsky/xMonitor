# xMonitor
======

xMonitor 是一个轻量,简单易用的监测图形分析工具。

### Dependencies

libevent RRDTool 

### Install



```bash
sudo apt-get install libevent
```

Then checkout the code and compile it
```bash
git clone https://github.com/0xsky/xmonitor
cd xmonitor
make
sudo make install
```
